# Simplicity Workspace

This is your Simplicity workspace directory. The AI assistant can read, write,
and manage files here when using tools like `read_file`, `write_file`, and
`list_directory`.

Drop any files you want the AI to work with here, or let the AI create
files for you during your chat sessions.

## Tips

- The AI can create, read, and edit files in this workspace
- Use `/save` in chat mode to save conversations to this directory
- Add custom tools to `~/.simplicity/tools/`
